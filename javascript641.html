Sample2

콜백함수를 이용한 오름차순 내림차순 예제입니다.

결과
내림차순 : 200,100,90,70,50,30,10,0
오름차순 : 0,10,30,50,70,90,100,200
html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>콜백 함수</title>
    <script>
        var num = [0,50,100,70,90,30,10,200];
        
        function func81(x,y){
            return y-x;
        }
        function func82(x,y){
            return x-y;
        }
        
        document.write("내림차순 : " + num.sort(func81),"<br>");
        document.write("오름차순 : " + num.sort(func82),"<br>");
    </script>
</head>
<body>
    
</body>
</html>
