continue문

break문과 continue문은 반복문을 제어하는 명령어입니다.

continue문

코드 실행 중에 continue을 만나면, 뒷 부분은 실행하지 않고 다음 반복 시퀀스로 넘어갑니다.

Sample : continue문 View
<!DOCTYPE html>
<html lang="ko">
    <head>
    	<meta charset="utf-8">
        <title>자바스크립트(JavaScript)</title>
        <script type="text/javascript">
			var count = 0;	
			while(true){
				count++;	
				
				if(count == 3){	/
					continue;
				}
				document.write("" + count + "번째 실행");
				document.write("<br />");
				
				if(count > 9){	
					break;
				}
			}
        </script>
    </head>
    
    <body>
	...
    </body>
</html>
