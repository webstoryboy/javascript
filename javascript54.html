증감 연산자

증가 연산자는 1만큼 증가시키고, 감소 연산자는 1만큼 감소시킵니다.

증감 연산자

연산자 표현 설명
++ x++ 또는 ++x x = x + 1
-- x-- 또는 --x x = x - 1

Sample1

대입 연산자를 이용한 예제입니다.

결과
9
10
20,21
22,22
html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>증감 연산자</title>
    <script>
        var num1 = 10;
        var num2 = 20;
        var result;

        num1--;     //9
        document.write(num1, "<br>");

        num1++;     //10
        document.write(num1, "<br>");

        result = num2++;    //20, 21
        document.write(result, "<br>");
        document.write(num2, "<br>");

        result = ++num2;    //22, 22
        document.write(result, "<br>");
        document.write(num2, "<br>");
    </script>
</head>
<body>
    
</body>
</html>
