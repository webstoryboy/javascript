날짜 객체

날짜 객체(Data Object)는 날짜와 관련된 객체입니다.

날짜 객체 생성

var 참조 변수 = new Data();

Date 함수
함수 설명
Date.now 현재 날짜 및 시간의 값을 리턴합니다.
Date.parse 날자를 포함한 문자열 구문을 분석하여 해당 날짜 사이의 시간을 밀리세컨드로 리턴합니다.
Date.UTC UTC(지역표준시) 또는 GMT(그리니치 표준시)의 날짜 사이의 시간을 밀리세컨트로 리턴합니다.
Date 메서드
메서드 설명
getDate() 로컬 시간을 사용하여 일(월 기준) 값을 리텁합니다.
getDay() 로컬 시간을 사용하여 일(주 기준)값을 리턴합니다.
getFullYear() 로컬 시간을 사용하여 연도 값을 리턴합니다.
getHours()> 로컬 시간을 사용하여 시간 값을 리턴합니다.
getMilliseconds() 로컬 시간을 사용하여 밀리세컨드 값을 리턴합니다.
getMinutes() 로컬 시간을 사용하여 분 값을 리턴합니다.
getMonth() 로컬 시간을 사용하여 월 값을 리턴합니다.
getSeconds() 로컬 시간을 사용하여 초 값을 리턴합니다.
getTime() 1970년 1월 1일 자정 이후로 밀리세컨드 단위로 리턴합니다.
getTimezoneOffset() 호스트 컴퓨터와 UTC(지역 표준시) 사이의 시간차를 분으로 리턴합니다.
getUTCDate() UTC를 사용하여 일(월 기준)값을 리턴합니다.
getUTCDay() UTC를 사용하여 일(일 기준)값을 리턴합니다.
getUTCFullYear() UTC를 사용하여 연도 값을 리턴합니다.
getUTCHours() UTC를 사용하여 시간 값을 리턴합니다.
getUTCMilliseconds() UTC를 사용하여 밀리세컨드 값을 리턴합니다.
getUTCMinutes() UTC를 사용하여 분 값을 리턴합니다.
getUTCMonth() UTC를 사용하여 월 값을 리턴합니다.
getUTCSeconds() UTC를 사용하여 초 값을 리턴합니다.
getVarData() Data 객체의 VT_DATE 값을 리턴합니다.
getYear()> 연도 값을 리턴합니다.
hasOwnProperty() 객체에 지정된 이름을 가진 속성이 있는지 여부를 나타내는 불린 값을 리턴합니다.
isPrototypeOf() 다른 객체의 prototype 체인이 있는지 여부를 나타내는 불린 값을 리턴합니다.
propertyIsEnumerable() 지정된 속성이 객체의 일부인지와 열거할 수 있는지 여부를 나타내는 불린 값을 리턴합니다.
setData() 로컬 시간을 사용하여 숫자로 된 날짜(월 기준)을 설정합니다.
setFullYear() 로컬 시간을 사용하여 연도 값을 설정합니다.
setHours() 로컬 시간을 사용하여 시간 값을 설정합니다.
setMilliseconds() 로컬 시간을 사용하여 밀리세컨드 값을 설정합니다.
setMinutes() 로컬 시간을 사용하여 분 값을 설정합니다.
setMonth()> 로컬 시간을 사용하여 월 값을 설정합니다.
setSeconds() 로컬 시간을 사용하여 초 값을 설정합니다.
setTime() Date 객체의 날짜와 시간 값을 설정합니다.
setUTCDate() UTC를 사용하여 숫자로 된 날짜(월 기준)을 설정합니다.
setUTCFullYear() UTC를 사용하여 연도 값을 설정합니다.
setUTCHours() UTC를 사용하여 시간 값을 설정합니다.
setUTCMilliseconds() UTC를 사용하여 밀리세컨드 값을 설정합니다.
setUTCMinutes() UTC를 사용하여 분 값을 설정합니다.
setUTCMonth() UTC를 사용하여 월 값을 설정합니다.
setUTCSeconds() UTC를 사용하여 초 값을 설정합니다.
setYear() 로컬 시간을 사용하여 연도 값을 설정합니다.
toDateString() 날짜를 문자열 값으로 리턴합니다.
toGMTString() GMT를 사용하여 문자열로 변환된 날짜를 리턴합니다.
toISOString() 날짜를 ISO 형식의 문자열 값으로 리턴합니다.
toJSON() JSON serialization 이전에 객체 형식의 날짜를 변환하는데 사용합니다.
