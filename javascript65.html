while문

var 변수 = 초기값
while (조건식){
    실행문;
    증감식;
}

Sample : while문
<!DOCTYPE html>
<html lang="ko">
    <head>
    	<meta charset="utf-8">
        <title>자바스크립트(JavaScript)</title>
        <script type="text/javascript">
			var count = 0;	
			while(true){
				count++;	
				document.write("" + count + "번째 실행");
				document.write("<br />");
				
				if(count > 9){	// 조건
					break;
				}
			}
        </script>
