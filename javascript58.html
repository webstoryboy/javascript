문자열 결합 연산자

문자열과 연산자를 결합해서 사용할 수 있습니다.

문자열 결합 연산자

Sample : 문자열 결합 연산자
<!DOCTYPE html>
<html lang="ko">
    <head>
    	<meta charset="utf-8">
        <title>자바스크립트(JavaScript)</title>
        <script type="text/javascript">
			var x = "java";
        	var y = 30;
        	
        	document.write(x + 100);  // 숫자와 연산
        	document.write("<br/>");
        	document.write(x + true); // boolean 타입과 연산  
        	document.write("<br/>");
        	document.write(x + null); // null과 연산 
        	document.write("<br/>");
        	document.write(x + "script"); // 문자열과 연산 
        	document.write("<br/>");
        	document.write(true + true); // boolean타입 연산(true는 1, false는 0으로 인식) 
			document.write("<br/>");
			document.write(true + null); // boolean타입과 null (null은 0으로 인식)
			document.write("<br/>");
			
			y = "" + y; // y를 문자열로 변환
			document.write(y + 30);
			
        </script>
    </head>
    
    <body>
	...
    </body>
</html>
