논리 연산자

참/거짓으로 나타나는 boolean 값을 조합해서 논리 연산을 수행합니다.

논리 연산자

연산자 예시 설명
&& X && Y (AND)둘다 true인 경우 true
|| X || Y (OR)둘 중의 하나 이상이 true이면 true
! !X (NOT)식이 false인 경우 true
Sample : 논리 연산자
<!DOCTYPE html>
<html lang="ko">
    <head>
    	<meta charset="utf-8">
        <title>자바스크립트(JavaScript)</title>
        <script type="text/javascript">
			var x = 10;
        	var y = 20;
        	var z = 100;
			
			document.write(true && true);
			document.write("<br/>");
			document.write(true && false);
			document.write("<br/>");
			document.write(true || true);
			document.write("<br/>");
			document.write(true || false);
			document.write("<br/>");
			document.write(!true);
			document.write("<br/>");
			
			//논리 연산자의 활용
			document.write(x < y && y < 120);
			document.write("<br/>");
			document.write(70 <= z && z < 120);
			document.write("<br/>");
			document.write(80 <= z && z <= 100);
			
        </script>
    </head>
    
    <body>
	...
    </body>
</html>
