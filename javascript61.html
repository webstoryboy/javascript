if ~ else문

두개의 값을 비교하여 코드를 별개로 실행합니다.

if ~ else문

if ~ else문은 두개의 조건이 있을 때 사용하는 조건문입니다.

if (조건식) {
    //참(true)일 때 실행 코드
} else {
    //거짓(false)일 때 실행 코드
}
