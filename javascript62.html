다중 if문

여러가지 조건에 따라 조건문을 설정합니다.

다중 if문

if (조건) {
    //참일 때 실행 코드
} else if {
    //앞 조건이 거짓일 때 실행 코드
} else if {
    //앞 조건이 거짓일 때 실행 코드
} else {
    //앞의 모든 조건이 거짓일 때 실행 코드
}
