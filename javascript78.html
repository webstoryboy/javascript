배열 객체

배열 객체(Array Object)는 여러 개의 데이터를 저장하기 위한 객체입니다.

배열 객체 생성

var 참조 변수 = new Array();

배열 객체
속성 설명
length 배열 요소의 개수를 알아냅니다.
배열 함수
함수 설명
Array.isArray 객체가 배열인지 여부를 나타내는 불린 값을 리턴합니다.
배열 메서드
함수 설명
concat() 두 배열의 조합으로 구성된 새 배열을 리턴한다.
every() 배열의 모든 요소에 대해 선언된 콜백 함수가 true를 리턴하는지 여부를 확인합니다.
filter() 배열의 각 요소에 대해 선언된 콜백함수를 호출하고 콜백 함수가 true를 리턴하는 값의 배열을 리턴합니다.
forEach() 배열의 각 요소에 대해 선언된 콜백 함수를 호출합니다.
hasOwnProperty() 객체에 지정된 이름을 가진 속성이 있는지 여부를 나타내는 불린 값을 리턴합니다.
indexOf() 배열에서 맨 처음 나오는 값의 인덱스를 리턴합니다.
isPrototypeOf() 객체가 다른 객체의 prototype 체인에 있는지 여부를 나타내는 불린 값을 리턴합니다.
join() 배열 하나에 함께 연결된 모든 요소들로 구성된 String 객체를 리턴합니다.
lastIndexOf() 배열에서 마지막으로 나오는 지정된 값의 인덱스를 리턴합니다.
map() 배열의 각 요소에 대해 선언된 콜백 함수를 호출하고 결과가 포함되어 있는 배열을 리턴합니다.
pop() 배열의 마지막 요소를 제거하여 리턴합니다.
propertyIsEnumerable() 지정된 속성이 객체의 일부인지와 열거할 수 있는지 여부를 나타내는 불린 값을 리턴합니다.
push() 배열에 새 요소를 추가하고 배열의 새 길이를 리턴합니다.
reduce() 배열의 모든 요소에 대해 선언된 콜백함수를 호출하여 단일 결과를 누적합니다.
reduceRight() 배열의 모든 요소에 대해 선언된 콜백 함수를 호출하여 단일 결과를 내림차순으로 누적합니다.
reverse() 요소들이 역으로 나열된 Array 객체를 리턴합니다.
shift() 배열에서 첫 번째 요소를 제거하여 리턴합니다.
slice()> 배열의 일정 부분을 리턴합니다.
some() 배열의 모든 요소에 대해 선언된 콜백 함수가 true를 리턴하는지 여부를 확인합니다.
sort()> 요소가 정렬된 Array 객체를 리턴합니다.
splice() 배열에서 첫 번째 요소를 제거하고, 필요하면 그 자리에 새 요소를 삽입한 다음 삭제된 요소를 리턴합니다.
toLocaleString() 현재 로케일을 사용하는 문자열을 리턴합니다.
toString() 배열을 나타내는 문자열을 리턴합니다.
unshift() 배열 시작에 새요소를 삽입합니다.
valueOf() 배열에 대한 참조를 가져옵니다.
