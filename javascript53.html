대입 연산자

대입 연산자는 사칙연산을 간결하게 표현할 때 사용합니다.

대입 연산자

연산자 표현 설명
+= x += 10 x = x + 10
-= x -= 10 x = x - 10
*= x *= 10 x = x * 10
/= x /= 10 x = x / 10
%= x %= 10 x = x % 10

대입 연산자를 이용한 예제입니다.

Sample2

대입 연산자를 이용한 테이블 만들기 예제입니다.

결과
1 2 3
html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>table</title>
    <script>
        var table = "<table border='1'>";
        table += "<tr>";
        table += "<td>1</td><td>2</td><td>3</td>";
        table += "</tr>";
        table += "</table>";

        document.write(table);
        console.log(table);
    </script>
</head>
