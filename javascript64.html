조건부 연산자

조건부 연산자는 조건을 처리하는 다른 형태의 제어문입니다.

조건부 연산자(삼항 연산자)

(조건) ? (참일 때 실행할 코드) : (거짓일 때 실행할 코드);

Sample1

if문을 삼항 연산자로 변경하는 예제입니다.

html
<!DOCTYPE html>
<html lang="ko">
<head>
	<meta charset="utf-8">
	<title>자바스크립트(JavaScript)</title>
	<script type="text/javascript">
		// if~else문
		let value = window.prompt("숫자를 입력해주세요.");
		value = parseInt(value);

		if (value % 2 == 0){
			document.write("짝수입니다.");
		} else {
			document.write("홀수입니다.");
		};
		
		//조건 연산자
		let value = window.prompt("숫자를 입력해주세요.","");
		value = parseInt(value);

		(value % 2 == 0) ? document.write("짝수입니다.") : document.write("홀수입니다.");;
	</script>
</head>
<body>

</body>
</html>
