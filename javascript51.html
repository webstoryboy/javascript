변수의 검사

현재 변수에 어떠한 자료형이 저장되어 있는지 검사합니다.

typeof(검사할 대상)

Sample : 변수의 검사 View
<!DOCTYPE html>
<html lang="ko">
    <head>
    	<meta charset="utf-8">
        <title>자바스크립트(JavaScript)</title>
        <script type="text/javascript">
        	var var1 = 30;	// 숫자 대입
			document.write(typeof(var1));
			document.write("<br />");
			
			var1 = "hello";	// 문자열 대입
			document.write(typeof(var1));
			document.write("<br />");
			
			var1 = function(){};	// funtion 대입
			document.write(typeof(var1));
			document.write("<br />");
			
			var1 = {};	// 객체 대입
			document.write(typeof(var1));
			document.write("<br />");
			
			var var2;	// 초기화 안함
			document.write(typeof(var2));
			document.write("<br />");
        </script>
    </head>
    
    <body>
	...
    </body>
</html>
