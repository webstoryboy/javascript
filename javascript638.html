재귀 함수

재귀함수는 함수 정의문 내에 함수를 다시 호출하여 실행하는 함수입니다. 함수를 반복문처럼 사용할 때 사용합니다.

function 함수이름(){
     //실행내용
     함수이름();
}
함수이름(); //함수호출
