산술 연산자

산술 연산자는 사칙 연산을 수행합니다.

산술 연산자

연산자 예시 설명
+ x + y 더하기
- x - y 뻴셈
* x * y 곱하기
/ x / y 나누기
% x % y 나머지
Sample : 산술 연산자
<!DOCTYPE html>
<html lang="ko">
    <head>
    	<meta charset="utf-8">
        <title>자바스크립트(JavaScript)</title>
        <script type="text/javascript">
        	var x = 10;
        	var y = 20;
        	
        	document.write(x + y);
        	document.write("<br />");
        	document.write(x - y);
        	document.write("<br />");
        	document.write(x * y);
        	document.write("<br />");
        	document.write(x / y);
        	document.write("<br />");
        	document.write(x % y);
        	
        </script>
    </head>
    
    <body>
	...
    </body>
</html>
