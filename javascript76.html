숫자 객체

숫자 객체(Number Object)는 숫자와 관련된 객체입니다. 자바스크립트를 이용하여 숫자를 표시하거나, 표현 가능한 수의 속성과 표기법에 대한 메서드를 제공합니다.

숫자 객체 생성

var 참조 변수 = new Number(값)

숫자 객체 메서드 및 속성
종류 설명
hasOwnProperty 객체에 지정된 이름을 가진 속성이 있는지 여부를 나타내는 불린 값을 리턴합니다.
isPrototypeOf 다른 객체의 prototype 계층 구조에 있는지 여부를 나타내는 불린 값을 리턴합니다.
propertyIsEnumerable 지정된 속성이 객체의 일부인지 그리고 열거할 수 있는지 여부를 나타내는 불린 값을 리턴합니다.
toExponential(n) 지수 표시로 나타낸 숫자를 포함하는 문자열을 리턴합니다.
toFixed(n) 숫자를 고정 소수점 표시로 나타내는 문자열을 리턴합니다.
toLocaleString 현재 위치를 기반으로 문자열로 변환된 객체를 리턴합니다.
toPrecision(n) 지수 표시나 고정 소수점 표시로 나타낸 숫자를 포함하고 지정된 자릿수를 포함하는 문자열을 리턴합니다.
toString() 객체를 나타내는 문자열을 리턴합니다.
valueOf 지정한 객체의 기본값을 리턴합니다.
