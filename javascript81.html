window 객체

window 객체는 DOM 구조의 최상위 객체입니다.

window 객체

window 객체 속성
종류 설명
open() 새로운 윈도우 창을 생성합니다.
closed 브라우저 창이 열려 있는지 닫혀 있는지에 대한 정보의 값을 반환합니다.
defaultStatus 브라우저 상태 표시줄 글자의 기본 값을 표현합니다.
document 하위 document 객체
frames 브라우저에서 사용된 모든 프레임을 배열 형태로 제공합니다.
history 하위 history 객체
innerHeight 브라우저의 콘텐츠 영역의 높이
innerWidth 브라우저의 콘텐츠 영역의 너비
length 브라우저에 사용된 프레임의 개수
location 하위 location 객체
name 브라우저 전체의 표시 이름
navigator 하위 navigator 객체
opener 열려 있는 창에 대한 레퍼런스
outerHeight 툴바/스크롤바를 포함한 바깥 영역의 높이
outerWidth 툴바/스크롤바를 포함한 바깥 영역의 너비
pageXOffset 브라우저의 문서 위치(스크롤된 위치)의 x좌표
pageYOffset 브라우저의 문서 위치(스크롤된 위치)의 y좌표
parent 현재 창의 부모창
screen 하위 screen 객체
screenLeft 브라우저 화면에서의 x좌표
screenTop 브라우저 화면에서의 y좌표
screenX 브라우저 화면에서의 x좌표
screenY 브라우저 화면에서의 y좌표
self 현재 사용 중인 창
status 브라우저의 상태 표시줄의 글자
top 제일 위에 표시된 브라우저 창
window 객체 메서드
종류 설명
alert() 경고창을 뛰웁니다.
blur() 현재 윈도우의 포커스를 없애줍니다.
close() 현재 윈도우를 닫습니다.
stop() 창이 로그되는 것을 멈춥니다.
confirm() ok, cancel 형태의 다이얼로그를 뛰어줍니다.
createPopup() 팝업 창을 만듭니다.
focus() 현재 윈도우로 포커스를 설정합니다.
moveBy() 윈도우 창의 위치를 상대적으로 이동합니다.
moveTo() 윈도우 창의 위치를 이동시킬 때 사용합니다.
open(URL, name, features, replace) 새창을 열어줍니다.
print() 현재 창의 콘텐츠 부분을 프린트합니다.
prompt() 사용자 입력이 가능한 입력창을 띄어줍니다.
resizeBy() 윈도우 창의 크기를 상대적으로 변경할 때 지정합니다.
resizeTo() 윈도우 창의 크기를 변경할 때 사용합니다.
scrollBy() 윈도우 스크롤의 위치를 상대적으로 이동합니다.
scrollTo() 윈도우 스크롤의 위치를 절대적으로 이동합니다.
setInterval() 일정 시간마다 함수를 반복해서 실행합니다.
setTimeout() 일정 시간 후 함수를 한 번만 실행합니다.
clearInterval() 일정 시간마다 함수를 반복하는 것을 중지합니다.
clearTimeout() 일정 시간 후 함수를 한 번 실행하는 것을 중지합니다.
